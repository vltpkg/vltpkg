Usage:

<Code
  code="$ vlt run-exec [command ...]"
  title="Terminal"
  lang="bash"
/>

If the first argument is a defined script in package.json, then this
is equivalent to `vlt run`.

If not, then this is equivalent to `vlt exec`.
