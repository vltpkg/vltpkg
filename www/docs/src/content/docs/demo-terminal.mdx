# Terminal Demo

A composite terminal component styled after the vlt expressive-code
frame: traffic-light dots, mono body, themed via the same design
tokens.

## Default size

<Terminal
  title="vlt — zsh"
  prompt="~/vlt ❯"
  lines={[
    { text: 'vlt install', type: 'input' },
    { text: 'added 142 packages in 1.2s', type: 'success' },
    { text: 'vlt run build', type: 'input' },
    { text: 'ERR missing script "build"', type: 'error' },
  ]}
/>

## Fixed width and height

Props `width` and `height` accept numbers (px) or strings (any CSS
unit). The body scrolls when content overflows the height.

<Terminal
  title="vlt — long output"

## Composed children

Use the subcomponents to build the body imperatively, including a
blinking cursor via `TerminalCursor`.

<Terminal title="vlt — interactive" prompt="~/vlt ❯">
  <div className="text-muted-foreground">
    <TerminalPrompt>~/vlt ❯</TerminalPrompt>{' '}
    <span className="text-foreground">vlt query ":missing"</span>
  </div>
  <TerminalOutput variant="success">
    1 missing dependency found
  </TerminalOutput>
  <TerminalOutput variant="error">
    ERR install required to reconcile
  </TerminalOutput>
  <div className="mt-1">
    <TerminalPrompt>~/vlt ❯</TerminalPrompt>
    <TerminalCursor />
  </div>
</Terminal>
