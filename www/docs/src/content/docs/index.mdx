# Documentation

VLT delivers the tools and infrastructure developers need to streamline package management, scale efficiently, and secure a faster, more tailored web experience.

## Getting Started

Learn how to get the VLT client set up in your project.

<Tabs>
  <TabItem label="npm">
    <Code code="$ npm i -g vlt" lang="bash" />
  </TabItem>
  <TabItem label="pnpm">
    <Code code="$ pnpm i -g vlt" lang="bash" />
  </TabItem>
</Tabs>

> vlt is supported on Node versions 20 or 22 currently.
