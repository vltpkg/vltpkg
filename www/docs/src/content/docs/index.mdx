# Documentation

vlt (/vōlt/ pronounced "volt") is a platform that delivers the tools
and infrastructure to meet the challenges of modern Javascript supply
chain management, so teams can ship and maintain packages more easily.

[vlt.io](https://vlt.io/onboarding-signup) serves as a drop-in
replacement for the npm registry.

## Getting Started

Learn how to get the VLT client set up in your project.

<Tabs>
  <TabItem label="npm">
    <Code code="$ npm i -g vlt" lang="bash" />
  </TabItem>
  <TabItem label="pnpm">
    <Code code="$ pnpm i -g vlt" lang="bash" />
  </TabItem>
</Tabs>

> vlt is supported on Node versions 20 or 22 currently.
