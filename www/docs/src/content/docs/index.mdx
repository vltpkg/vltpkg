## Introduction to vlt

vlt is the next generation platform for Javascript registries.

Our toolset streamlines package management for a secure, faster,
developer experience.

vlt was started by seasoned developers of the core npm team and
veteran open source contributors who have seen the challenges of
critical package distribution at scale.

To learn more how we're different, check out
["Why Vlt?"](/get-started/why-vlt)

## Using vlt packages

vlt has every package that npm already has so you still have access to
all the libraries you're used to.

All you have to do:

<Steps>
  <ol>
    <li> Set your package manager to use vlt as the registry</li>
    <li> Run install</li>
  </ol>
</Steps>

## Publishing a package to vlt

### Sign Up

To publish a private or public package with vlt, you first have to
[sign up](https://www.vlt.io/onboarding/account-setup). You can sign
up with  
Github, Google (Gmail) or with an email and password.
