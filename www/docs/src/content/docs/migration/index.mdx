### Pre-requisites

This tutorial assumes working familiarity with npm, git, and running
bash commands within your computer's terminal.

1. If you haven't already:
   [Sign up](https://www.vlt.io/onboarding/account-setup)

<Badge text="Protip" variant="tip" /> Hang onto your username; you're
gonna need it!

<Tabs>
  <TabItem label="pnpm">

1. Assuming you have a package you want to publish, grab the project
   path from its root.

<Code
  title="Switch vlt to be your registry"
  code={`pnpm config set registry https://registry.vlt.io/<your-username>/npm --location=project-path`}
/>
  </TabItem>
  <TabItem label="yarn">
    <Code
      title="Switch vlt to your registry"
      code={`{echo 'registry "https://registry.vlt.io/your-username/npm/" >> .yarnrc'}`}
    />
    <Code
      title="login to obtain your auth token"
      code={`yarn login --registry https://registry.vlt.io/<your-username>/main`}
    />
    <Code
      title="Switch all dependencies to vlt's npm mirror"
      code="rm -rf yarn.lock node_modules && yarn install"
      />
  </TabItem>
  <TabItem
    label="yarn berry"
    >
   <Code
      title="Set yarn config to use the vlt.io npm mirror"
      code={`yarn config set npmRegistryServer "https://registry.vlt.io/your-username/npm/" --location=project`}
    />
    This will update the `.yarnrc.yml`.

<Code
  title="Log in to obtain an authentication token"
  code="yarn npm login --web-login --always-auth"
/>

<Code title="Publish your package" code="yarn publish" />

Your .yarnrc.yml will look like this:

```yaml
npmScopes:
  <your-username-scope>:
    npmAlwaysAuth: false
    npmAuthToken: '${NODE_AUTH_TOKEN:-}'
    npmRegistryServer: 'https://registry.vlt.io/your-username/npm/'
```

<Code
  title="Switch all dependencies to vlt's npm mirror"
  code="rm -rf yarn.lock node_modules && yarn install"
/>

</TabItem>
