### CLI Options

| Option     | Alias | Default | Description             |
| ---------- | ----- | ------- | ----------------------- |
| `--port`   | `-p`  | `1337`  | Server port             |
| `--config` | `-c`  | -       | Config file path        |
| `--debug`  | `-d`  | `false` | Debug mode              |
| `--daemon` | -     | `true`  | Local filesystem daemon |
| `--help`   | `-h`  | -       | Show help               |
